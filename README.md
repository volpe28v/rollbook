# RollBook 

ATND風のイベント管理ツールです。
ATNDみたいな素敵なツールで、社内のイベントを管理できたらいいなと思って作りました。

## Installation

```bash
$ git clone git@github.com:volpe28v/rollbook.git
$ cd rollbook
$ bundle
$ rake db:migrate
$ rails s
```
